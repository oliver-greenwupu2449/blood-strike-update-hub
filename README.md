## Overview

Blood Strike ESP Hack is built for players who want better awareness during matches without changing game files. It places important match data directly over the game view, making it easier to follow enemy movement, watch health status, judge distance, and spot useful loot in real time.

The tool is designed to stay lightweight and responsive, so it can run smoothly on mid-range systems while preserving stable frame rates. For both solo play and coordinated team matches, the added on-screen context can support faster reactions and more informed decisions in firefights.

## How to Use

After the program starts, an overlay will appear above your Blood Strike window. The default hotkeys make it easy to switch individual features on and off:

- **F1** - Toggle Player ESP on/off
- **F2** - Toggle Health Bars
- **F3** - Toggle Loot ESP
- **F4** - Open Color Customizer menu

To change settings, right-click the system tray icon and choose "Configuration." Any edits are applied right away, so there is no need to restart the game.

## Configuration

The settings file is stored as plain text at:

`blood-strike-esp-hub-pc\config.ini`

Open it in any text editor to adjust:

- Overlay opacity and scale
- Key bindings for each feature
- Color values for player ESP, health bars, and loot markers
- Distance measurement units (meters/feet)

Example configuration block:

```ini
[Display]
Opacity=0.85
Scale=1.0

[Colors]
PlayerESP=#00FF00
HealthBar=#FF0000
LootHighlight=#FFFF00
```

## FAQ

**Will anti-cheat detect this tool?**  
The overlay works outside the game process and does not inject code into the client. Even so, use it at your own discretion, since all game enhancement tools involve some level of risk.

**What is the update process for a newer release?**  
Grab the newest build and overwrite the files in your current folder. If you leave `config.ini` in place, your settings will remain intact.

**Can the overlay be moved around?**  
Yes. You can drag overlay elements with the mouse to reposition them, and the layout is saved automatically.

**What if nothing appears on screen?**  
Make sure the tool is running as Administrator and that Blood Strike is using windowed or borderless windowed mode. Also check whether your antivirus is preventing the overlay from starting.
